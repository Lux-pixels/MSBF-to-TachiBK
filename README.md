# MSBF-to-TachiBK

Convert Manga Storm (.msbf) exports into Komikku/Mihon (.tachibk) backups.

## Features

- Import Manga Storm exports
- Convert to Komikku backups
- Drag-and-drop desktop interface
- Cross-platform (Windows, Linux, macOS)
- Open source (MIT)
