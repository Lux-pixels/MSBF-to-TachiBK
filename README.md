# MSBF-to-TachiBK

Convert **Manga Storm (.msbf)** exports into **Komikku/Mihon (.tachibk)** backups.

MSBF-to-TachiBK is an open-source converter that allows Manga Storm users to migrate their manga library into modern Tachiyomi-based applications such as **Komikku**.

---

## Features

- ✅ Parse Manga Storm `.msbf` backup files
- ✅ Convert to Komikku `.tachibk` backups
- ✅ Preserve manga titles and MangaDex links
- ✅ Automatically map MangaDex source IDs
- 🚧 Desktop drag-and-drop interface (planned)
- 🚧 Windows, Linux and macOS support
- 🚧 Open source (MIT License)

# Building

Requirements:

- Java 17
- Gradle Wrapper (included)

Build:

```bash
./gradlew clean build
```

Run:

```bash
./gradlew run --args="samples/favorites.msbf"
```

Or specify an output filename:

```bash
./gradlew run --args="samples/favorites.msbf MyBackup.tachibk"
```

# Restoring into Komikku 1.13.6


## Before Restoring

Before restoring the generated backup:

1. Install the **MangaDex** extension inside Komikku.
2. **Disable Delegated Sources** in Komikku.
3. If using **BlueStacks**, copy the generated `.tachibk` file into Android's **Downloads** folder before restoring.
4. Do **not** restore directly from the BlueStacks shared folder (`BstSharedFolder`), as Android may not be able to access the file.

### Disable Delegated Sources

In Komikku, go to:

```
Settings
    → Browse
        → Delegated Sources
```

Turn **Delegated Sources** **OFF** before restoring the backup.

This ensures the MangaDex extension is used directly and prevents source resolution issues during the restore process.

## Restore

Open:

```
Settings
    → Backup & Restore
        → Restore Backup
```

Select the generated backup.

---

## IMPORTANT

For the current development version (**v0.1**), restore **Manga only**.

Uncheck the following options:

- ☐ Saved Searches
- ☐ Source Settings
- ☐ Extension Repos
- ☐ App Settings
- ☐ Categories

Leave **Manga** checked.

These sections are not currently generated by the converter and restoring them may overwrite existing Komikku settings or produce unnecessary warnings.

## Known Issues

Current limitations:

- MangaDex extension must already be installed.
- Reading history is not yet restored.
- Chapter progress is not yet restored.
- Tracking information is not yet restored.
- Categories are not yet restored.
- URL normalization is still being refined.
