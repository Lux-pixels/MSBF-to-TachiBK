# Restore Guide

This guide explains how to restore a backup generated by **MSBF-to-TachiBK** into Komikku.

---

## Before You Restore

Before restoring the generated `.tachibk` file:

1. Install Komikku.
2. Install the MangaDex extension.
3. Generate the backup with metadata enabled.
4. Copy the `.tachibk` file onto your Android device.
5. If using BlueStacks, move the `.tachibk` file into Android’s **Downloads** folder before restoring.

Metadata is enabled by default. Use `--no-metadata` only for fast testing.

---

## Recommended Restore Options

When restoring into Komikku, check:

```text
Manga
Categories
App Settings
```

Uncheck:

```text
Saved Searches
Source Settings
Extension Repos
```

---

## Why App Settings Should Be Checked

The generated backup currently includes one app setting:

```text
enable_delegated_sources = false
```

This disables delegated sources during restore.

This is included because delegated MangaDex behavior can cause restored MangaDex entries to open with a WebView / website 404 error instead of opening correctly through the MangaDex source.

---

## Why Metadata Should Be Included

MangaDex entries restore more reliably when metadata is included.

The converter attempts to add:

```text
Title
Author
Artist
Description
Genres
Cover URL
Manga status
```

Metadata is fetched automatically unless `--no-metadata` is used.

---

## BlueStacks Restore Notes

If Komikku cannot open the backup from the shared BlueStacks folder, copy the `.tachibk` file to:

```text
Android Downloads
```

Then restore from that location inside Komikku.

This avoids file access errors caused by restoring directly from a shared host folder.

---

## Recommended Test Restore

For testing, use a clean Komikku profile when possible.

Recommended restore checklist:

```text
Install MangaDex extension
Generate backup with metadata
Move backup into Android Downloads
Restore Manga
Restore Categories
Restore App Settings
Do not restore Source Settings
Do not restore Extension Repos
Do not restore Saved Searches
Open several MangaDex entries
Confirm categories restored
Confirm MangaDex pages open without WebView / 404 errors
```

---

## Current Known Restore Limitations

Currently not restored:

```text
Chapters
Reading progress
Reading history
Tracking data
Saved searches
Source settings
Extension repos
```

These may be added after V1 if the Manga Storm backup contains enough data to support them.
